## 🧩 Features

- 🔐 Generates Mega full creds.json for **any Baileys bot**
- ☁️ Stores creds securely with **MEGA**
- 📱 Web-based **Code Pair Pairing**
- 🚀 One-click deploy to:
  - Heroku
  - Render
  - Koyeb
  - Self-hosting platforms
